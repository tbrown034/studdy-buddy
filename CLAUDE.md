# Study Buddy - Coding Study Platform

## Project Overview

Personalized coding study platform with interactive lessons, quizzes, and flashcards for JavaScript, TypeScript, React, and Next.js.

**URL**: studdy-buddy-rho.vercel.app
**GitHub**: github.com/tbrown034/study-buddy

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Interactive coding lessons
- Quizzes with immediate feedback
- Flashcard system for key concepts
- Progress tracking
- Topics: JavaScript, TypeScript, React, Next.js
